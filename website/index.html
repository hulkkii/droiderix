ANDROID-INTEGRATED AI AGENT

Agent-powered control of your Android.

Control your device through structured workflows and natural language. Droiderix runs agent tasks inside local and remote workspaces, executes commands through shell and tools, and automates workflows with Android background services.

Initialize Access Explore Capabilities

Core Architecture

Built around workspace-scoped execution, persistent sessions, and Android-native service hooks instead of vague autonomy claims.

Workspace Architecture

Operate inside local app-data workspaces or configured SSH workspaces, each with persistent chat sessions and scoped file access.

Shell & SSH Access

Execute commands through Termux integration for local shell tasks or keep remote SSH terminal sessions active with foreground service support.

Pluggable AI Backends

Choose between local, Codex, and GitHub Copilot agent runtimes depending on the workspace and task.

Scheduled Automation

Launch agent runs through scheduled jobs and background services, with persisted run state and user-visible notifications while work continues.

Local-First Persistence

Chats, runs, schedules, and workspace metadata are stored on-device with SQLite and secure storage rather than depending on a constant cloud session.

System Integrations

Connect workflows to Android notifications, calendar access, file tools, git operations, and mini-app runtime features inside the existing app host.
