# Droiderix

Droiderix is an Android app for agent-powered workflows, mini-app generation, workspace-based execution, and local or remote tooling from your phone.

This repository does not contain the Droiderix application source code.

It exists for two public purposes:

- Static marketing website for Droiderix
- Bug reports, issue tracking, and feature requests from users

## What To Use This Repo For

- Report bugs you found in Droiderix
- Request features or improvements
- Follow public website updates

## What Is Not In This Repo

- Private Droiderix app source code
- Internal implementation details beyond what is shared publicly

## Website

The marketing site lives in [`website/`](./website) and is configured for GitHub Pages deployment.
